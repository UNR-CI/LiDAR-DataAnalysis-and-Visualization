About

Purpose

The LiDAR Data Visualization Tool is a highly adaptable software that creates visualizations based on user needs. This application was developed with the sole purpose to visualize and analyze LiDAR point cloud data being collected from sensors located on street lights throughout the city of Reno, Nevada. Currently, there are two active sensors located directly outside the University of Nevada, Reno campus on the corner of Virginia and 15th Street with plans to continue to expand the sensor network throughout the city. Through visualization and analysis of the data, insight and readability can be obtained and increased.

Guide to Use App

This application is designed to take input from the user in the data explorer form where the user selects the sensor location, data visualization type, and data variables such as count and statistics. Once these values are selected and submitted, the user will be able to view the data visualization dashboard which will contain the data analysis based on the inputted values. The main goal is to be able to use this visualization tool to analyze traffic patterns as well as any other useful information.
